# Storage Module

Storage abstraction for creating, updating, deleting and browsing files (plus more) supporting:

- Amazon S3
- Rackspace Cloud Files
- EMC Atmos
- FTP
- Local File System

## Getting Started

Recommend starting out with Storage documentation locally using the User Guide module.
